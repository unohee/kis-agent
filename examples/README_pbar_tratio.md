## 🎯 매물대 분석이란?

매물대(Volume Profile)는 특정 가격대에서 거래된 총 거래량을 보여주는 분석 도구입니다.

### 핵심 개념
- **매물대**: 특정 가격대에서 많은 거래량이 발생한 구간
- **지지선**: 하락 시 매수세가 몰리는 가격대 (현재가 > 매물대)
- **저항선**: 상승 시 매도세가 몰리는 가격대 (현재가 < 매물대)

### 활용 방법
1. **최대 매물대 확인**: 가장 거래량이 많은 가격대
2. **현재가와 비교**: 이격률 계산으로 위치 파악
3. **신호 해석**: 지지/저항 가능성 판단

## 📊 결과 해석 가이드

### 신호별 의미
- **SUPPORT (지지)**: 현재가가 매물대 위에 있어 하락 시 지지받을 가능성
- **RESISTANCE (저항)**: 현재가가 매물대 아래에 있어 상승 시 저항받을 가능성  
- **NEUTRAL (균형)**: 현재가가 매물대와 근접하여 중요한 변곡점

### 이격률 해석
- **양수 (+)**: 현재가가 매물대보다 높음 (상승 추세)
- **음수 (-)**: 현재가가 매물대보다 낮음 (하락 추세)
- **절댓값이 클수록**: 매물대에서 많이 벗어난 상태

## ⚠️ 주의사항

### API 제한사항
- **호출 빈도**: 연속 호출 시 0.1~0.2초 간격 유지 권장
- **정산 시간**: 23:30~01:00 사이에는 일부 API 사용 제한
- **장 운영 시간**: 시장 개장 시간에 가장 정확한 데이터 제공

### 투자 참고사항
- **보조 지표**: 매물대는 참고용 보조 지표로 활용
- **다른 분석과 병행**: 기술분석, 기본분석과 함께 종합 판단
- **투자 책임**: 모든 투자 결정은 개인의 책임

### 결과 데이터 활용
테스트 결과는 JSON 파일로 저장되므로 추가 분석에 활용할 수 있습니다.

```python
import json

# 저장된 결과 로드
with open('pbar_tratio_test_results_20250115_143025.json', 'r') as f:
    results = json.load(f)

# 데이터 분석
for result in results:
    if result['success']:
        code = result['code']
        signal = result['comparison']['signal']
        gap_pct = result['comparison']['gap_percentage']
        print(f"{code}: {signal} ({gap_pct:+.1f}%)")
```
